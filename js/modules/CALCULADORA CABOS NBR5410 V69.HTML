Ajuda e referências (fórmulas e parâmetros considerados)

1) Corrente de projeto (Ib)

  • Se informado “Corrente (A)”, então: Ib = corrente informada.
  • Se informado “Potência”:
    • Trifásico: kSys = √3 ; Monofásico: kSys = 1.
    • Demanda / Folga / Projeto: Pcalc = P · (Demanda/100) · (1 + Folga/100) · Fator de projeto
    • Se kVA: Ib = (S[kVA]·1000) / (kSys · V)
    • Se kW: Ib = (P[kW]·1000) / (kSys · V · cosφ · η)

2) Correções térmicas e agrupamento

  • fT: fator de temperatura ambiente (interpolação linear na tabela interna do programa).
  • fG: fator de agrupamento (conforme modo de agrupamento selecionado).
  • fTotal = fT · fG

3) Corrente corrigida requerida

  • Iz_requerida = Ib / fTotal
  • Cabos em paralelo: nPar (a capacidade total é multiplicada por nPar; R e X são divididos por nPar).

4) Critério de capacidade de condução (ampacidade)

  • Seleciona-se a menor seção que atende: Iz_tabela · nPar ≥ Iz_requerida
  • Tabela usada na seleção atual: —

4.1) Harmônicas no neutro (N)

  • Critério por faixa ou por valor medido (h3%) (selecionado pelo usuário).
  • < 15%: neutro pode ser reduzido (somente quando marcada a condição “circuito equilibrado + neutro protegido + mesmo material”).
  • 15% a 33%: adota N = S e considera o conjunto como 4 condutores carregados com fNC = 0,86.
  • > 33%: considera 4 condutores carregados com fNC = 0,86 e dimensiona o neutro com FH: FH = 1,15 (até 66%) ou FH = 1,73 (acima de 66%).

4.2) Bitola travada (cliente/projeto recebido)

  • Quando “Bitola travada” está ativada, a seção final é fixada pelo usuário e o programa não faz seleção automática.
  • O programa recalcula a verificação em duas frentes:
    • Ampacidade: Imax_amp = Iz_total (da bitola travada) · fTotal (considerando paralelos).
    • Queda de tensão: Imax_vd é obtida pela equação inversa de ΔV% no limite informado.
  • A potência máxima exibida é baseada em Imax = min(Imax_amp, Imax_vd).

5) Queda de tensão (ΔV%)

  • L_km = L(m)/1000
  • Monofásico (2 condutores no circuito): k = 2
  • Trifásico: k = √3
  • ΔV = k · I · (R·cosφ + X·senφ) · L_km
  • ΔV% = (ΔV / V) · 100
  • Limite considerado: ΔV% ≤ limite informado

6) Resistência e reatância consideradas

  • R20 (Ω/km) = (ρ20 · 1000) / S
  • R(θ) = R20 · [1 + α · (θ − 20°C)]
  • X (Ω/km): obtida por seção na tabela interna (XL_BY_SECTION). Se não existir para a seção, usa-se X_fallback = — Ω/km.
  • Constantes internas:
    • ρ20 cobre = — Ω·mm²/m, α cobre = — 1/°C
    • ρ20 alumínio = — Ω·mm²/m, α alumínio = — 1/°C

7) Dimensionamento de Neutro (N) e Terra/Proteção (PE)

  • Neutro (N): regra simplificada pela Tabela 3 do Guia (NBR 5410):
    • Se S ≤ 25 → N = S
    • Seções tabeladas: 35→25; 50→25; 70→35; 95→50; 120→70; 150→70; 185→95; 240→120; 300→150; 400→185
    • Se S for maior que 400 mm² (ou não estiver na tabela), por conservadorismo o app adota N = arredonda_para_cima(S/2) usando as seções padronizadas disponíveis no próprio app.
  • Terra (PE): regra da Tabela 4 do Guia (NBR 5410):
    • Se S ≤ 16 → PE = S
    • Se 16 < S ≤ 35 → PE = 16
    • Se S > 35 → PE = arredonda_para_cima(S/2) usando as seções padronizadas disponíveis no app.
  • Paralelos: por padrão o app assume a mesma quantidade de cabos em paralelo informada para as fases (× nPar) também para N e PE, apenas para apresentação. Se você desejar, pode ajustar manualmente no memorial final para o projeto.
